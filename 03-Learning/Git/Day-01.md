* What is Git?
Distributed VCS that helps developers track their codebases, collaborate with others and manage multiple versions of project.

* Repository
A repository is a centralised storage location where things are kept, organised, and maintained.A project with memory

* Working Directory
A place where you are actually editing files.

* Staging Area
A place where you add your work from working directory. Think of a packaging box where you add all things and pack it. Similarly this the place where the things packed inside the box is committed to the local repository.

* Commit 
A permanent checkpoint 

* git status
displays the current state of the repository, including tracked and untracked files, staged changes, and unstaged modifications.

* git add
moves changes from the Working Directory to the Staging Area so they can be included in the next commit.

* git commit
Make a commit

* git log
Look at branch's history

* git diff
Show difference between two commits
