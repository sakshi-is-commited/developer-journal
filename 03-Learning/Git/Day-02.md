* what is a Git Object?
- Git object consists of BLOB, TREEE and COMMIT

* What is a BLOB?
- Contents of a file (No metadata)

* What is a TREE?
- A directory listing (Of Blobs and Trees)

* What is a Commit Object?
- A commit object stores the author, timestamp, commit message, parent commit(s), and a pointer to the tree object representing the project's state.

* What is a SHA-1 Hash?
- It consists of 20 bytes usually represented by 40 characters in hexadecimal forms

* What is HEAD?
- A special pointer in git which basically points to the current branch we are actually on.

* What is a Branch Pointer?
- HEAD.
HEAD points to a branch.

The branch points to a commit.

So a branch pointer is not HEAD.

HEAD is a special pointer that usually points to the current branch.
